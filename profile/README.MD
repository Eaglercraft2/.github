# Eaglercraft2

## Info:
A library of voxel games all made with JavaScript, TypeScript, Python, and HTML.  
It uses a custom game engine called **ELGE (Eaglercraft2 Low-End Game Engine)** and a custom game renderer called **Victus**.  
The frontend is built with React and Vite, and the project is currently in development.
